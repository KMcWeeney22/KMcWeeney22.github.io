Built with: HTML / CSS / JavaScript / Jquery / Bootstrap / Youtube Data Api

View Code
View Live

Concept

A homepage for the TV show The Walking Dead where fans can come and find related news and videos all in one place.

Purpose

The purpose of this project was to take create a website utilizing my HTML, CSS and JavaScript Skills. I thought it would fun to build a website for The Walking Dead because it is one of my favorite tv shows at the moment and I had a pretty good idea of how I wanted to build it and what third party APIs and other data I was going to include. The API I used was Googles YouTube data API. Which I called the last 8 videso that were uploaded to AMCs The Walking Dead playlist on YouTube. I also created a feed reader widget from a website that collects the recent news about The Walking Dead and also added a Twitter Widget that displays the latest Tweets with the hastag #thewalkingdead. The layout structure of the page was created using Bootstrap.
